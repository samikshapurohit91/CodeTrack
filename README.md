# CodeTrack 

CodeTrack Pro is a modern coding profile tracker that allows users to search and analyze Codeforces profiles. It provides real-time statistics, rating history visualization, favorite profiles, search history, and performance analytics through an interactive dashboard.

# Features

-  Search Codeforces Users
-  Profile Dashboard
-  Current Rating & Max Rating
-  Rank Information
-  Rating History Chart
-  Save Favorite Profiles
-  Search History
-  Dark / Light Mode
-  Download Profile Report
-  Local Storage Support
-  Fully Responsive Design

# Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Codeforces API
- LocalStorage
